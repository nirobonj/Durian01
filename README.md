
# Durian01
ขั้นตอนการใช้งาน
1. Server
Command dev
    installing
        - pip install django
        - pip install djangorestframework
        - pip install psycopg2
        - pip install -r requirements.txt
    models
        - python manage.py makemigrations <API APP name>
        - python manage.py migrate
    run app api
    - python manage.py runserver

2. Flutter
Command dev
    - flutter pub get
    - flutter run
    
Version
Python 3.10.0
Dart SDK version: 3.4.1
